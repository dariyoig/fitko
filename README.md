# Fitko

## Project Idea
Fitko is an AI-powered fitness system that creates custom workout plans, features an interactive  AI chatbot coach, and recommends real personal trainers to help users reach their goals.

## Purpose
- Chat with users to learn about their fitness needs
- Generate personalized workout plans
- Suggest real trainers for extra support

## Stack
- Frontend: React (TypeScript)
- Backend: Python (FastAPI/Flask)
- Database: MongoDB/PostgreSQL
- Authentication: Auth0/Firebase Auth
- AI: OpenAI GPT-4 API
